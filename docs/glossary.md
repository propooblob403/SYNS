# SYNS Glossary

## Signal event
A typed local observation such as a trade, wallet move, LP shift, or code update.

## Synaptic node
A narrow detector with limited focus and a configurable threshold.

## Adaptive weighting
The accumulation of score as repeated evidence overlaps with node sensitivity.

## Convergence
The degree to which multiple nodes become active inside the same scenario.

## Emergence
A protocol-level outcome produced when convergence and weighted evidence cross threshold together.
